# AI Resume Screening & Job Matching System

A Flask-based web application that compares resumes with job descriptions using NLP techniques.

## Features
- Upload Resume (PDF)
- Paste Job Description
- Calculates Match Score using TF-IDF & Cosine Similarity
- Displays Missing Skills

## Tech Stack
- Python
- Flask
- Scikit-learn
- PDFPlumber

## How to Run

```bash
pip install -r requirements.txt
python app.py
